Introduction

TankCoders is a 3D tanks battle simulator where each tank is conceived as an AgentSpeak(L) agent.

To run the agents written in AgentSpeak(L) code, the Jason BDI engine is used. More informations about this engine can be found here.

Project Information

This section provides a quick overview of the project development and the free tools and libraries used.

The TankCoders project was conceived as Final Project of the Computer Science graduation to the Universidade Regional de Blumenau (FURB) in 2008-1. This project was developed by academic Germano Fronza and oriented by teacher Dalton Solano dos Reis. The monograph can be downloaded here.

Tools and Libraries Overview

Type Tool/Library name Description
Library Jason A Java-based interpreter for an extended version of AgentSpeak.
Library JMonkey Engine High performance game library providing all the tools needed to build quality 3D games and applications in Java.
Library LWJGL Java Native Interface (JNI) to bind OpenGL with Java.
Library JME Physics 2 An extended version of OdeJava to integrate with JMonkeyEngine.
Library FengGUI Provides UI components to OpenGL applications.
Library Java Game Networking Abstract library to write multiplayer games in Java.
Tool Blender Light-weight tool to 3D modeling.
Tool Assembla Space Project management tools: Wiki, Trac, Developers, SVN, Tickets and Milestones.
