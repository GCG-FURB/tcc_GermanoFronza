Frequently Asked Questions

  1. How can I get help?
  2. How can I generate TankCoders JAR?
  3. How can I run the Jason projects in examples folder?
  4. How can my agents reflect the terrain topografy?

1. How can I get help?

The best way to get help is sending an e-mail to Germano Fronza.

2. How can I generate TankCoders JAR?

  1. Open the comand line console and move to the TankCoders installation directory
  2. Run command: $>ant jar
  3. The generated JAR will be put in lib directory with the name tank-coders-{version}.{release}.jar

3. How can I run the Jason projects in examples folder?

  1. Open the comand line console and move to an example project folder in TankCoders installation directory
  2. Run command: $>ant getRequirements
  3. This ant task generates the folders lib and rsrc used by Jason Project to run TankCoders simulator

4. How can my agents reflect the terrain topografy?

  1. To do that, your first task is to create an Jason Internal Action
  2. After do it, you can retrieve the Height Map Adapter instance with the line (static method):

    Terrain.getHeightMapAdapter(); // you can call me, I'm static :)

  3. With the Height Map Adapter instance you are able to retrieve all the relevant informations about the terrain topography
