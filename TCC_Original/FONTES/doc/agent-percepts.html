Agent Perceptions

All the agents that have defined the class TankAgArch as architecture class will have the following perceptions during the battle:

  • +onStartBattle

    Notified when the battle starts.

  • +onScanTank(TankName, IsEnemy, PosX, PosY, PosZ)

    Notified when scans another tank in radar.

    Arguments:

    1. String TankName - Tank name scanned
    2. Integer IsEnemy - 0 if friend, 1 if is enemy
    3. Float PosX - Position X of scanned tank location
    4. Float PosY - Position Y of scanned tank location
    5. Float PosZ - Position Z of scanned tank location

  • +onHitTank(TankName, IsEnemy)

    Notified when hits another tank.

    Arguments:

    1. String TankName - Tank name
    2. Integer IsEnemy - 0 if friend, 1 if is enemy

  • +onBulletHit(TankName, IsEnemy)

    Notified when your fired bullet hits another tank.

    Arguments:

    1. String TankName - Tank name affected
    2. Integer IsEnemy - 0 if friend, 1 if is enemy

  • +onHitByBullet(BulletPower)

    Notified when your tank has took a shot.

    Arguments:

    1. Integer BulletPower - Destruction power of the bullet

  • +onBulletHitBullet

    Notified when your fired bullet hits another bullet.

  • +onBulletMissed

    Notified when your fired bullet misses.

  • +onDeath

    Notified when your tank dies.

  • +onTankDeath(TankName)

    Notified when another tank dies.

    Arguments:

    1. String TankName - Tank name dead

  • +onTeamWin

    Notified when your team wins.
