Getting Started with TankCoders

This section aims to help you install and run TankCoders, as well as developing a simple multi-agent system using Jason to build a tanks team.

Download and installation

  1. Java JDK >= 6.0 is required (http://java.sun.com);
  2. Apache Ant is required (http://ant.apache.org/);
  3. Download TankCoders from http://www.assembla.com/spaces/tankcoders
  4. Create a folder for your TankCoders installation and decompress the distribution file in it;
  5. Done! It's that simple!

Running

First of all you need to execute the Game Server:

  1. Open the comand line console and move to the TankCoders installation diretory;
  2. Run the command line: $>ant runServer
  3. Configure arguments of the game and then Start.

For client application, there are two ways to run:

  • Avatar Player: Run this way when you want to control a tank manually via keyboard. Steps:
    1. Open the comand line console and move to the TankCoders installation diretory;
    2. Run the command line: $>ant runClient

  • MAS Player: Run this way when you want your agents to control the tanks in the battle. Steps:
    1. If not created yet, create a new Jason Project (using jEdit-based version or eclipsePlugin);
    2. Enter the TankCoders installation directory and copy the folders rsrc and lib to the root directory of the created Jason project;
    3. In mas2j project file, define the environment class: br.furb.inf.tcc.tankcoders.jason.TankCodersEnvironment;
    4. For each agent, define the agentArchClass: br.furb.inf.tcc.tankcoders.jason.TankAgArch;
    5. Execute the Jason project. An error will be thown, but don't worry! follow the steps normally;
    6. Inside the directory bin, Jason generates the ant file build.xml. Make a copy of this file and rename it to c-build.xml;
    7. Open the file c-build.xml and find the ant task compile. Inside this task exists an tag called javac, after this, include this source:

      <copy todir="${build.dir}"/>
            <fileset dir="${basedir}/rsrc"/>
      </copy todir="${build.dir}">

    8. Continue in c-build.xml file and find the ant task run. Inside this task exists an tag called java, within this, include the tag:

            <jvmarg line='-Djava.library.path="lib/native"'/>

      after the already existent tag:

            <jvmarg line="-Xmx750M"/>

    9. Now you can run the Jason project normally.
