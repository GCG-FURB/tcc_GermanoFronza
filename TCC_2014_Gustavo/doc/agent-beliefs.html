Agent Beliefs

All the agents that have defined the class TankAgArch as architecture class will have the following beliefs during the battle:

  • info(PosX, PosY, PosZ, Health, MainGunBullets, ChassiHeading, MainGunHeading)

    This belief is added each N milliseconds, where N is configured in the Game Configuration File. The default value is 500 ms.

    Arguments:

    1. Float PosX - Position X of the tank
    2. Float PosY - Position Y of the tank
    3. Float PosZ - Position Z of the tank
    4. Integer Health - Current health of the tank
    5. Integer MainGunBullets - Current main gun of the tank
    6. Float ChassiHeading - Current chassi heading in degrees
    7. Float MainGunHeading - Current main gun heading in degrees

  • actionFinished

    This belief is added when an action finish. Example: execute the action ahead(40), when the tank reaches this 40 meters then this belief is added.
